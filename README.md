# 代码架构入门教程

> 写给普通人的代码架构入门，学会架构思维，避开 vibecoding（靠感觉写代码）的陷阱。

## 📖 简介

很多初学者写代码是这样的：
- 想到什么写什么
- 全部代码堆在一个文件里
- 复制粘贴凑功能
- 凭感觉调试，报错就改一改试试
- 过了一个月自己都看不懂

这就是 **vibecoding**，短期内能跑起来，但长期维护会让你痛苦不堪。

本教程从零开始，一步步教会你**正确的代码架构思维**，每一点都带详细示例和错误对比，看完就能用。

## ✨ 特点

- 🎯 **面向普通人**：不需要你有多年经验，从零基础开始讲
- 🎯 **每一点都有示例**：概念 + ❌错误写法 + ✅正确写法 + 常见坑对比
- 🎯 **解决实际问题**：告诉你为什么错，正确该怎么做
- 🎯 目标：**教会普通人基础的代码架构，避免 vibecoding 犯错**

## 🎯 适合谁

- 刚入门编程，不知道怎么组织代码
- 一直写业务代码，想学习架构思维
- 想改掉"凭感觉写代码"的坏习惯
- 准备面试，温习基础架构知识

## 📖 阅读方式

1. 按顺序从第一章读到最后一章
2. 每个概念看完，看示例，对比错误和正确写法
3. 动手实践，在你的项目里用起来
4. 记住检查清单，写代码前过一遍
